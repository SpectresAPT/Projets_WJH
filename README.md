## 🧾 Description
Weather Judgment Hub (WJH) est une application qui permet de suivre la météo de deux façons :  
1. **Récupération automatique des données météo via l'API OpenWeather**.  
2. **Saisie manuelle des informations météo** pour compléter ou corriger les données.  

Le projet est conçu pour **apprendre et expérimenter le développement en C++ et QT**, tout en manipulant des APIs et des interfaces graphiques.


---

## 💻 Fonctionnalités
- Récupération des données météo depuis une API externe  
- Interface graphique simple avec QT  
- Saisie manuelle pour compléter ou corriger certaines données  

---
## ⚙️ Prérequis
Avant de compiler et exécuter WJH, assurez-vous d’avoir :  
- **Tout OS compatible avec Qt**
- **Qt 5 ou Qt 6** installé avec Qt Creator  
- **qmake** et **make** disponibles dans le terminal  
- **Clé API OpenWeather** pour récupérer automatiquement les données météo (optionnel pour usage manuel)
